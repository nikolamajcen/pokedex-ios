# Pokedex
*iOS application written as MVC.*

This application contains:
* Pokedex data
  * Pokemon search 
  * Pokemon details (image, description, evolution chain, stats)
* QR Code Catcher
  * Catch pokemon by QR code
* Trainer data
 
*Optional TODO: Add pokemon images in all 3 sizes.*
